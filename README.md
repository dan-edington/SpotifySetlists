# Spotify Setlists
Create Spotify playlists using setlist.fm data

http://www.danedington.com/spotifySetlists

### Instructions

Start APIServer in src/server :
```sh
node index.js
```

Start development build:
```sh
npm start
```

Start production build:
```sh
npm run build
```
